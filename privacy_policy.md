# Privacy Policy

**Date:** Feb 17, 2026

I am writing this for my very first app which is going to be on Play Store. It will be published by my friend known as ENDERMANYK.

For the app "Accessibility Button as Screenshot" or "Accessibility Screenshot":
- It does not use internet connection, not even request the permission in AndroidManifest.xml (which is necessary for internet usage)
- It does not collect, store, upload any user data
- It does not have any kind of telemetry
- It requests accessibility permission because it is used to trigger the screenshot
- It does not do anything other than registering action for the accessibility button which appears on system navigation bar or as floating icon
- If you have doubt, you're welcome to check the app code here: https://github.com/faizul726/accessibility-button-as-screenshot
